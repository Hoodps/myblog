自从使用git做为版本管理软件以来，会经常是使用到git的一些命令。同时为了方便以后查阅，就把一下常用的命令记录下来。

git init 初始化git仓库

git add . 

git commit -m 'first commit' 提交并且加注释

git push origin master 将文件给推到主干master

git push origin gh-pages 将文件给推到分支gh-pages
